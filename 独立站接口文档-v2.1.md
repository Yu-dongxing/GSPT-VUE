# 文章控制器

## POST 发布文章接口

POST /api/article/publish

文章状态和文章分类都使用数字；
 DRAFT(0, "草稿"),
    PUBLISHED(1, "已发布");

> Body 请求参数

```json
{
  "id": 0,
  "title": "string",
  "content": "string",
  "previewContent": "string",
  "coverFileId": 0,
  "coverImageUrl": "string",
  "previewImageId": 0,
  "previewImageUrl": "string",
  "imageWall": [
    {
      "fileId": 0,
      "imageUrl": "string",
      "sortOrder": 0
    }
  ],
  "status": "DRAFT",
  "category": "DEMAND"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|[ArticlePublishRequest](#schemaarticlepublishrequest)| 否 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": {
    "id": 0,
    "title": "string",
    "content": "string",
    "previewContent": "string",
    "coverFileId": 0,
    "coverImageUrl": "string",
    "previewImageId": 0,
    "previewImageUrl": "string",

      {
        "id": 0,

        "imageUrl": "string",
        "sortOrder": 0
      }
    ],
    "authorId": 0,
    "authorName": "string",
    "status": "DRAFT",
    "category": "DEMAND",
    "createTime": "string",
```
